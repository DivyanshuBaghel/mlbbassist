# Changelog

All notable changes to this project will be documented in this file.

## [1.1] - 2025-12-22

### Added
- **Smart Overlay Hiding**: The overlay now automatically hides itself for a split second when capturing the screen to ensure the analysis screenshot is clean and free of obstruction.
- **Manual API Key Selection**: Users can now manually select which of the 3 stored API keys to use from the main menu.
- **Improved Error Handling**: Better user feedback when API limits are reached or image capture fails.

### Fixed
- Fixed an issue where the overlay would block the view of the scoreboard during analysis.
- Fixed black screen artifacts that appeared on some devices during screen capture.
- Improved API key rotation logic.

## [1.0] - 2025-12-21
- Initial Release
- Basic overlay functionality
- Gemini AI integration for hero counters and strategy
- Multi-key support
